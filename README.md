# Downshift

Downshift is an iOS app for building custom driving routes, racing your own lines, and tracking progress over time.

## What It Does

- Create routes with a start, checkpoints, and finish using an interactive map.
- Run timed races with automatic checkpoint and finish detection.
- Follow rally-style live navigation while racing.
- Save results and review route performance with stats views.
- Explore social screens for friends and activity feed progress.

## Project Status

Active development. Core route creation and racing flows are implemented, with social and stats features continuing to expand.
